Detailed Description

Object class for Event.

Object class is either Cube::Handle<Cube::ReconTrack> or Cube::Handle<Cube::ReconCluster> + pdg, parent pdg and parentId information
It has both of the two reconstructed object as a member but only one of them is activated when the Object class is initialized.
